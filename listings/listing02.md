Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Функция test:
Имеет именованное возвращаемое значение x.
Внутри функции x присваивается значение 1.
Затем defer вызывает анонимную функцию, которая увеличивает x на 1. Но этот вызов отложен до завершения выполнения функции test.
При выходе из функции test, отложенная функция увеличивает x на 1, делая его равным 2.
Возвращает 2.

Функция anotherTest:
Имеет неименованное возвращаемое значение.
x присваивается значение 1.
defer откладывает вызов функции, которая увеличивает x на 1.
Возвращает значение x (которое в этот момент равно 1) и только после этого отложенная функция увеличивает x, но изменённое значение x не влияет на возвращаемое значение.
Возвращает 1.
```

Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
// Объявляем массив 'a' с 5 целочисленными элементами
    a := [5]int{76, 77, 78, 79, 80}

// Создаём срез 'b', который ссылается на элементы массива 'a'
// начиная с индекса 1 и заканчивая индексом 4 (не включая его)
    var b []int = a[1:4]

// Выводим срез 'b'. Он будет содержать элементы массива 'a' с индексами 1, 2, 3,
// то есть значения 77, 78, 79
    fmt.Println(b)
```
